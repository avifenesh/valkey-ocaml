Module Valkey.Cluster_router

Cluster-aware Router.t implementation.

Discovers topology from seeds via quorum, opens a Connection.t per node, and returns a Client.Router.t that dispatches by slot with Read_from honoured. MOVED/ASK retry and periodic refresh land in follow-up commits.
